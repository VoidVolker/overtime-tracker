Dev

Start server:

rails server

Start coffee-script watcher (webpacker coffee loader is broken):

coffee -c -b -M -w ./

Node requirements: v16. For >v16 (due webpacker bugs):

export NODE_OPTIONS=--openssl-legacy-provider
